### Review Notes

**IMPORTANT - Please Read Before Testing:**

**App Functionality:**
Craig-O Clean Terminator Edition is a system utility that requires admin privileges for some operations. The app is designed as a menu bar-only application (no Dock icon).

**Getting Started:**
1. Launch the app - it will appear in the menu bar (top-right)
2. Click the menu bar icon to access all features
3. The main window can be opened via "Open Main Window" in menu
4. Some cleanup operations will request admin password

**Features to Test:**

1. **Quick Cleanup** (Menu Bar ‚Üí Quick Cleanup)
   - Purges inactive memory
   - Clears caches
   - No admin password required for basic cleanup

2. **Full Cleanup** (Menu Bar ‚Üí Full Cleanup)
   - Deep system cleanup
   - May request admin password
   - Safe for daily use

3. **Process Management** (Main Window ‚Üí Processes tab)
   - View all running processes
   - Sort by CPU/Memory usage
   - Terminate processes (requires confirmation)

4. **Browser Management** (Main Window ‚Üí Browsers tab)
   - View tabs across Safari, Chrome, Firefox
   - Close resource-heavy tabs
   - Clear browser caches

5. **Diagnostics** (Main Window ‚Üí Diagnostics tab)
   - View real-time system metrics
   - Health score calculation
   - Performance recommendations

6. **Optional: Apple Sign In** (Settings ‚Üí Account)
   - Sign in with Apple ID
   - Sync settings across devices via iCloud
   - Upload profile avatar

7. **Optional: AI Features** (Settings ‚Üí AI)
   - Requires Ollama installation (optional)
   - Can be tested without Ollama
   - Instructions provided in-app

**Admin Privileges:**
The app requests admin privileges ONLY when necessary:
- Memory purge operations
- System cache cleanup
- DNS flush operations
- Never for basic monitoring or viewing data

**Safety:**
- System-critical processes are protected from termination
- All operations respect macOS System Integrity Protection (SIP)
- Comprehensive logging for transparency
- No data collection or telemetry

**Autonomous Mode:**
Can be enabled in Settings ‚Üí Automation
- Monitors system in background
- Triggers cleanup automatically
- Respects user-defined thresholds

**Testing Environment:**
- Tested on macOS 13.0+ (Ventura, Sonoma, Sequoia)
- Apple Silicon (M1/M2/M3/M4) and Intel Macs
- Works with or without Ollama installed

**Known Limitations:**
- Some browsers must be running for tab management
- First cleanup may request admin password
- iCloud sync requires signed-in Apple ID on device

**Privacy & Security:**
- All processing happens locally
- No external servers contacted (except iCloud for sync if enabled)
- No analytics or tracking
- Open source core components
