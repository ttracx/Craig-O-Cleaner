## üõ†Ô∏è Build & Upload Commands

### Archive Build (Xcode)
```bash
# Clean build
xcodebuild clean -project CraigOTerminator.xcodeproj -scheme CraigOTerminator

# Create archive
xcodebuild archive \
  -project CraigOTerminator.xcodeproj \
  -scheme CraigOTerminator \
  -configuration Release \
  -archivePath ./build/CraigOTerminator.xcarchive
```

### Upload to App Store Connect

**Option 1: Via Xcode**
1. Product ‚Üí Archive
2. Window ‚Üí Organizer
3. Select archive
4. Click "Distribute App"
5. Select "TestFlight & App Store"
6. Follow prompts

**Option 2: Via Transporter**
1. Export as .pkg from Xcode
2. Open Transporter app
3. Drag .pkg file
4. Click "Deliver"

**Option 3: Via Command Line**
```bash
xcrun altool --upload-app \
  -f CraigOTerminator.pkg \
  -u YOUR_APPLE_ID \
  -p @keychain:AC_PASSWORD
```
