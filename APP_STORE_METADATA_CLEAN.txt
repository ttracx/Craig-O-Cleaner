1. LAUNCH AND PERMISSIONS
   - Launch the app
   - Grant Accessibility permission when prompted (System Settings > Privacy & Security > Accessibility)

   - These permissions are required for process monitoring and memory optimization

2. MAIN DASHBOARD
   - View real-time CPU, memory, and disk metrics
   - Observe live updates every 2 seconds
   - Check memory pressure indicator (Normal/Warning/Critical)

3. MEMORY OPTIMIZATION
   - Navigate to Memory tab
   - Click "Analyze Memory" to scan for cleanup candidates
   - Review suggested apps for cleanup
   - Click "Smart Cleanup" to free memory (safe - only terminates background apps)

4. PROCESS MANAGEMENT
   - Navigate to Processes tab
   - View all running applications with memory usage
   - Sort by memory or CPU usage
   - Test terminating a non-critical process

5. BROWSER TAB MANAGEMENT
   - Open Safari or Chrome with multiple tabs
   - Navigate to Browser tab in Craig-O-Clean
   - View all open tabs
   - Select tabs and close them
   - Verify tabs are closed in browser

   - Click Craig-O-Clean icon in menu bar

   - Navigate between Dashboard, Memory, Browser, and Settings tabs
