# Agent Quick Reference

## Commands

### Code Quality
```
@code-reviewer REVIEW [file]
@code-reviewer SECURITY_SCAN [file]
```

### Testing
```
@test-generator GENERATE_TESTS [file]
@test-generator EDGE_CASES [file]
```

### Documentation
```
@doc-generator DOCUMENT [file]
@doc-generator README [project]
@doc-generator OPENAPI [api]
```

### Refactoring
```
@refactor-assistant ANALYZE [file]
@refactor-assistant SIMPLIFY [file]
```

### Security
```
@security-auditor AUDIT [file]
@security-auditor OWASP_CHECK [file]
```

### Performance
```
@performance-optimizer ANALYZE [file]
@performance-optimizer QUERY_OPTIMIZE [sql]
```

### iOS/macOS
```
@swiftui-expert IMPLEMENT [feature]
@swiftui-expert VIEW [component]
```

### VibeCaaS
```
@vibecaas-branding BRAND_COMPONENT [component]
@vibecaas-branding THEME_TOKENS
```

### Quantum
```
@quantum-algorithm-expert DESIGN_ALGORITHM [problem]
@quantum-algorithm-expert VQE_SETUP [hamiltonian]
```

### Orchestration
```
@agent-orchestrator ORCHESTRATE [complex_request]
```
